# Davidium Browser

# DOWNLOAD IT AT: [Google Drive Link](https://drive.google.com/drive/folders/1K-8ZS5GhSgxvQa3WFHV_y7nrPIlpst2U?usp=sharing)



Davidium Browser is a custom Python browser built with PyQt5 and QtWebEngine.  
It features a custom home page, a Google search bar inside the app, and a sleek GUI similar to Chrome/Firefox.

---

## Features

- Embedded Google search inside the app  
- Customizable home page (`home.html`)  
- Lightweight and portable  

---

## Getting Started

1.Download the **Davidium** folder
2. Make sure `home.html` and `_internal` are in the same folder as `Davidium.exe`.  
3. Run `Davidium.exe` to launch the browser.

---

## Files

- `Davidium.exe` — the browser executable  
- `home.html` — the home page of the browser
- `_internals` - the **NEEDED** files for the app to run
